# ZGWebImage
仿写SDWebImage
# 基本实现SDWebImage的功能，如：
# 1，普通图片
# 2，gif图片
# 3，图片内存缓存，文件缓存（硬盘缓存）
# 4，UIImageView UIButton 分类
# 5，placeholder占位图处理，completeBlock回调等
